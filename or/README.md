# or

Пакет для объединения нескольких `done`-каналов в один.
Возвращаемый канал закрывается, когда закрывается **любой** из входных.


## Сборка

```bash
go build ./...
```

Запуск примера:

```bash
go run .
```

---

## Тесты

Запуск всех тестов:

```bash
go test ./...
```

Запуск с подробным выводом:

```bash
go test -v ./...
```
